Markdown Highlighter

The highlighter in core/highlighter.py applies lightweight styling to Markdown constructs and highlights SRS-IDs using a configurable regex. It operates on a QTextDocument and focuses on clarity over completeness.

Highlights

  • Headings (H1–H6) with increasing sizes and spacing.
  • Inline code and fenced code blocks.
  • Lists, tables, and blockquotes with subtle borders.
  • SRS-IDs are emphasized to aid commit message preparation.

Customization

The stylesheet used by the preview is small and intentionally readable. Extend it to match your organization’s style or to emphasize specific sections (e.g., risks, NFRs).
