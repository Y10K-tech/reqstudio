Core Package

The core package implements non-UI functionality: Git operations, Markdown highlighting, templates, and utility helpers. UI code in app.py depends on these modules.

Modules

  • git_backend.py — High-level Git facade (open/init, commit, branches, history, push/pull).
  • highlighter.py — Markdown + SRS-ID highlighting for QTextDocument.
  • templates.py — Built-in templates in Swedish and English.
  • utils.py — Helpers like detect_srs_ids and newline normalization.
  • git_extensions/ — Extension points for additional Git-related commands (future).
