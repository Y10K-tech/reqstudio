Configuration

The desktop app stores lightweight preferences via QSettings, such as your last opened workspace. Additional configuration will evolve with features like template packs and validation rules.

Current Settings

  • Workspace: Last opened folder path.
  • View mode: Editor vs. Preview toggle (session-level).

Planned

  • Template packs (JSON/YAML) with organization overrides.
  • SRS-ID validation policies and commit guards.
  • Theme preferences (light/dark, fonts, spacing).
