CLI

A command-line interface is scaffolded under core/cli. The main entrypoint is intended to be rscli.py, with usage like rscli [COMMAND]. Commands will help automate tasks such as template generation, validation, exports, and integration with CI.

Status

  • Scaffold present: core/cli/rscli.py, core/cli/readme.md.
  • Implementation is pending; see Roadmap.

Planned Commands

  • rscli init — Initialize a requirements workspace.
  • rscli template <TYPE> — Emit a template (HL, LL, CMP, ADR, Ticket).
  • rscli validate — Check SRS-ID format and references.
  • rscli export <path> — Batch export to PDF/HTML.
  • rscli link — Scan and link IDs across files (report).
