FAQ

Why Git for SRS?

Plain text + Git makes requirements auditable, reviewable, and easy to integrate with developer workflows.

Does it support rich diagrams?

Use Markdown-compatible diagrams (e.g., Mermaid in code blocks) or link external diagram files; richer export is on the roadmap.

How do I enforce SRS-IDs?

Include IDs in document headers and commit messages. Future plugins will provide validation and commit guards.

Is there a web version?

The focus is a desktop app for performance and local control; APIs will support automation.
