UI Guide

The PyQt6 interface centers on fast editing with traceability. The main window toggles between the source editor and a rendered preview, includes a status bar for context, and provides Git-aware actions.

Layout

  • Editor: Plain Markdown/Text editor with a monospace font.
  • Preview: Read-only rendered Markdown view (single-pane when active). Uses a renderer that supports Markdown Extra and Pygments highlighting when available.
  • Status Bar: Workspace path, current file, branch, and detected SRS-IDs.
  • Menus/Toolbar: File operations, Git actions, template insertion, and view toggles.

Editing

  • Markdown headings, emphasis, code blocks, lists, tables, and blockquotes are styled for readability.
  • Use templates to scaffold SRS content and standardize sections.
  • Find (Ctrl+F) supports quick searches within the current document.

Git Actions

  • Stage/commit all or selective paths; default message can include SRS-IDs.
  • Create/switch branches; push/pull from the configured remote.
  • Per-file history view: navigate commits and open any version inline.

Preview and Export

  • Toggle preview to evaluate rendered content quickly.
  • Export to PDF using the Qt print pipeline.
