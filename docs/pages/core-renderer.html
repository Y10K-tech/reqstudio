HTML Renderer

The renderer in core/renderer.py turns Markdown into HTML for the preview pane and PDF export pipeline. It supports Markdown Extra (tables, fenced code) and syntax-highlighted code blocks via Pygments when available.

Rendering Pipeline

  • Preprocesses custom tags: {color:...}...{/color} and {highlight:...}...{/highlight}.
  • Uses markdown with extra and codehilite extensions when installed.
  • Falls back to a safe <pre> block when Markdown/Pygments are not present.
  • Provides a helper to emit Pygments CSS for dark or light themes.

APIs

render_markdown_html(text: str, css: str) -> str
  # Returns a full HTML document string; include your CSS for preview/export.

pygments_css(dark: bool) -> str
  # Returns CSS for the .codehilite class to style highlighted code.

Custom Tags

{color:red}Important{/color}
{highlight:yellow}Note{/highlight}

These are transformed into inline span elements with corresponding CSS inline styles.
