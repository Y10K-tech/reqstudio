App (PyQt6)

The main window binds together the editor, preview, status bar, and Git actions. It uses QSettings to remember the workspace and wires signals for menu/toolbar actions.

Main Components

  • Editor: QTextEdit with monospace font; bound to a Markdown highlighter.
  • Preview: QTextEdit read-only, rendering Markdown with a simple CSS stylesheet.
  • Status Bar: Displays workspace, current file, branch, SRS-IDs.
  • Git: GitFacade powers commit/branch/history/push-pull.

Dialogs

  • CommitDialog: prompts for a commit message (pre-filled with detected SRS-IDs when available).

Actions

  • Open/Save files, Open/Init workspace
  • Toggle preview
  • Insert template (HL/LL/CMP/ADR/Ticket)
  • Commit current/all, Branch ops, Push/Pull
  • View file history and open past versions
