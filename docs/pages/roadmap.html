Roadmap

Highlights from the public roadmap (subject to change):

  • Richer Markdown → styled PDF (pandoc/WeasyPrint optional integration)
  • SRS-ID live validation (highlight invalid formats)
  • Commit message guard to enforce SRS-ID presence
  • Branch naming wizard (e.g., feature/{SRS-ID}-{short})
  • Template manager (JSON/YAML) and org-wide packs
  • Plugin hooks (pre-commit checks, export transforms, linters)
  • Dark theme, font and spacing preferences
  • Localization packs (sv, en, …) for UI text
  • Unit tests (pytest) and CI badges
  • App bundle builds (PyInstaller) with signed binaries
