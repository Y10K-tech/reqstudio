Install

ReqStudio targets Python 3.10+ and uses PyQt6 for the desktop UI. Install from source via pyproject.toml, or use the platform scripts under install/. Installers attempt headless Python (and Git) installation when missing.

Prerequisites

  • Python 3.10 or later
  • Git (for workspace repos and GitPython)
  • Platform Qt dependencies (PyQt6 wheels handle most cases)

Setup (from source)

# 1) Create and activate a virtual environment
python -m venv .venv
. .venv/bin/activate     # Windows: .venv\\Scripts\\activate

# 2) Install project via pyproject.toml
python -m pip install --upgrade pip
python -m pip install .

# Optional (if defined): development extras
# python -m pip install .[dev]

# 3) Run the app
python app.py

Platform Installers

  • Linux/macOS: install/unix/install_reqstudio.sh — creates .venv, installs the project, adds a launcher at ~/.local/bin/reqstudio. Attempts to install Python/Git if missing.
  • Windows: install/windows/install_reqstudio.ps1 — creates .venv, installs, adds Start Menu/Desktop shortcuts; headless Python install via winget/choco/python.org if needed.
  • Windows Inno Setup: install/windows/inno/setup.iss — builds an installer that runs the PowerShell installer post-copy.
  • Windows PyInstaller: install/windows/pyinstaller/build_pyinstaller.ps1 — builds a self-contained ReqStudio.exe.
  • macOS .app bundle: install/macos/create_app_bundle.sh — creates ReqStudio.app that launches via .venv/bin/pythonw.

Docs Server (optional)

  • FastAPI server: python -m api.docs_server (default http://127.0.0.1:8808/docs/).
  • Docker: build and run install/docker/Dockerfile or docker-compose.yml.

Notes

  • If you hit dependency resolution issues, check pyproject.toml versions and extras.
  • On macOS/Linux, you may need additional Qt libraries provided by the wheel.
  • To build distributables (future), consider PyInstaller bundling.
