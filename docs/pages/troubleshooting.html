Troubleshooting

PyQt6 Installation Issues

  • Upgrade pip and retry: python -m pip install --upgrade pip.
  • Install from requirements.txt if pyproject.toml resolution fails.

Git Errors

  • Repo not open: Open a workspace or initialize a new Git repo.
  • No changes to commit: Stage changes before committing.
  • No remote configured: Add a remote before push/pull: git remote add origin <URL>.

PDF Export Problems

  • Try a different path or filename; ensure write permissions.
  • Consider printing to PDF from the OS print dialog as a fallback.

Encoding/Text Issues

  • Use UTF-8 files; normalize line endings (LF) if diffs look odd.
  • See normalize_newlines utility behavior for consistency.
