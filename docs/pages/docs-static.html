Docs Static Build

In addition to the single-page app (SPA) at /docs/, you can generate a classic multi-page static site from the same content. The static site lives under docs/site/ and each page is a standalone HTML file with a shared header and sidebar.

Build

python docs/build_static.py
# Output: docs/site/

Open

  • File:// — open docs/site/index.html directly in a browser (no server needed).
  • Server — when using the FastAPI docs server, visit /docs-site/.
  • Direct links — navigate to pages like /docs-site/overview.html.

Notes

  • The static site copies required media assets (logo) into docs/site/media/.
  • It reuses the same page content under docs/pages/ to avoid duplication.
  • The SPA provides search; the static site focuses on reliable multi-page navigation.
