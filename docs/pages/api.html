API & MCP

ReqStudio includes API scaffolding for two integration modes:

  • FastAPI server — Programmatic access to workspaces, documents, and export/validation (planned).
  • MCP integration — For LLM/workflow orchestration tooling (recommended with self-hosted models).

Notes

  • Use a self-hosted model for MCP; avoid third-party LLM providers for sensitive data.
  • The api/ folder contains placeholders for HTTPS, MCP, and pluggable backends.

Examples (future)

# FastAPI docs server (static docs)
python -m api.docs_server
# Default: http://127.0.0.1:8808/docs/

# MCP tool invocation (example sketch; subject to change)
mcp run reqstudio --op="validate" --path docs/*.md
