Template Library

core/templates.py provides template dictionaries for both Swedish and English. Types include HL, LL, CMP, ADR, and Ticket. Each template includes a title and a multi-section content body designed to be pasted into Markdown documents.

Localization

  • Swedish: BASE_TEMPLATE_SV — uses headings like Sammanfattning, NFR, Risker, Arkitektur.
  • English: BASE_TEMPLATE_ENGLISH — uses headings like Summary, Scope, API/Data, NFRs, Risks.

Extensibility

Consider externalizing templates to JSON/YAML for team-wide customization and versioning. The roadmap includes a template manager and organization-level packs.
