Docs Server

Run a minimalist FastAPI backend to serve the static documentation and media locally, offline.

Install

python -m venv .venv
. .venv/bin/activate                 # Windows: .venv\\Scripts\\activate
python -m pip install --upgrade pip
python -m pip install .

# Optional (if defined):
# python -m pip install .[dev]

Run

# Option 1: module entrypoint
python -m api.docs_server

# Option 2: uvicorn directly (default port 8808)
uvicorn api.docs_server:app --host 127.0.0.1 --port 8808

# Optional environment overrides
DOCS_HOST=0.0.0.0 DOCS_PORT=8080 python -m api.docs_server

# Open in your browser
# http://127.0.0.1:8808/docs/

Endpoints

  • / — Redirects to /docs/
  • /docs/ — Serves the documentation SPA
  • /media/ — Serves repository media (e.g., reqstudio_logo.png)
  • /healthz — Health check ({"ok": true})

Notes

  • Docs are served directly from the repository’s docs/ folder.
  • The SPA uses hash routing; static files are resolved under /docs/.
  • Media is mounted at /media to support the logo and assets referenced by the docs.
